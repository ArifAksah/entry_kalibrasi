

File postman_collection.json created successfully!
Isi: Authentication endpoints only (Login + Get Session)

Untuk endpoint lainnya, silakan add manual di Postman.

Cara import:
1. Buka Postman > Import > File > pilih postman_collection.json
2. Set Environment variables:
   - base_url: http://localhost:3000
   - supabase_url: https://your-project.supabase.co
   - supabase_anon_key: your-anon-key
   - token: (kosong, isi setelah login)

3. Run 'Login (Supabase Auth)' dulu, copy access_token ke variable 'token'
4. Sekarang endpoint lain yang butuh Bearer token akan otomatis pakai tokennya.
